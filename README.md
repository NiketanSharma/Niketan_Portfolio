## ✨ Highlights

| Feature | Details |
|---|---|
| **Interactive 3D Hero** | Embedded Spline 3D robot character that reacts to mouse movement |
| **Smooth Scrolling** | Lenis-powered smooth scroll synced with GSAP ScrollTrigger |
| **Cinematic Animations** | Stagger-reveal text, parallax project cards, scroll-driven scale/fade |
| **Preloader** | Animated loading screen with a clean transition into the main content |
| **Project Showcases** | Auto-playing video demos with parallax hover overlays |
| **Certifications Marquee** | Infinite-scroll carousel with click-to-expand modal |
| **Working Contact Form** | Email delivery via FormSubmit — no backend needed |
| **CV Download** | One-click resume download straight from the hero section |
| **Fully Responsive** | Optimised layouts for mobile, tablet, and desktop |

---

## 🛠️ Tech Stack

- **Framework** — React 18
- **Build Tool** — Vite 5
- **Animations** — Framer Motion · GSAP + ScrollTrigger
- **Smooth Scroll** — Lenis
- **3D Graphics** — Spline (embedded iframe)
- **Styling** — CSS Modules (scoped, zero-conflict)
- **Typography** — Plus Jakarta Sans (Google Fonts)
- **Form Handling** — FormSubmit (AJAX)

## 🚀 Getting Started

### Prerequisites

- **Node.js** ≥ 18
- **npm** ≥ 9 (or yarn / pnpm)

### Installation

```bash
# Clone the repository
git clone https://github.com/NiketanSharma/Niketan_Portfolio.git
cd Niketan_Portfolio

# Install dependencies
npm install

# Start the dev server
npm run dev
```

The app will be available at **http://localhost:5173**.

### Production Build

```bash
npm run build
npm run preview   # preview the production build locally
```
